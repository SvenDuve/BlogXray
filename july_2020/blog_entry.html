AI will be the new standard in X-Ray detection

We took a batch of approximately 5000 images of x-ray screens of children in China. The images were provided by Kaggle. The first issue we detected was class-imbalance. There were many more bacterial infections than viral ones. What happens is that if a certain class is much bigger than the other, the algorithm finds optimal solutions for predicting a certain class.

Now this can be overcome by making sure classes are balanced. So if we classify three types of images, we require roughly 33 per cent of each class. Once this is achieved, the algorithm is not likely to fall over to a certain class.

Class balance can be generated if we use the existing image base and fake new ones from old picture, by slightly tilting of applying a a very subtle coloring to them.
